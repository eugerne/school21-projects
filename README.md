# Школа 21 | Data Science Projects

Добро пожаловать в мой репозиторий с проектами, выполненными в рамках образовательной программы **Школы 21**.

Здесь собраны практические работы, посвященные изучению **Python**, **анализа данных**, **статистики** и **машинного обучения**. Каждый проект представляет собой самостоятельное исследование или практическую задачу, направленную на закрепление теоретических знаний и освоение современных инструментов Data Science.

Основная цель этого репозитория — показать мой путь обучения: от базовых алгоритмов и обработки данных до первых моделей машинного обучения.

# Описание проектов

## 01. Python Fundamentals

Первый проект посвящен изучению языка Python и его применению для обработки текстовой информации.

В ходе работы были реализованы:

- работа с виртуальными окружениями;
- использование Jupyter Notebook;
- очистка и токенизация текста;
- построение инвертированного индекса;
- поиск по тексту;
- вычисление TF–IDF для ранжирования документов.

**Полученные навыки**

`Python` • `OOP` • `Text Processing` • `Information Retrieval` • `TF-IDF`

---

## 02. NumPy Fundamentals

Проект посвящен библиотеке NumPy и работе с многомерными массивами.

Выполненные задачи:

- сравнение производительности Python и NumPy;
- написание собственных декораторов;
- решение головоломки Sudoku;
- тестирование алгоритмов;
- применение сингулярного разложения (SVD) для сжатия изображений;
- решение прикладных задач линейной алгебры.

**Полученные навыки**

`NumPy` • `Linear Algebra` • `Optimization` • `Testing` • `SVD`

---

## 03. Exploratory Data Analysis

Проект посвящен проведению разведочного анализа данных с использованием библиотеки Pandas.

В ходе проекта выполнялись:

- загрузка и объединение данных;
- очистка датасета;
- обработка пропусков;
- анализ распределений;
- группировка данных;
- построение сводных таблиц;
- визуализация результатов;
- поиск закономерностей в данных.

**Полученные навыки**

`Pandas` • `EDA` • `Data Cleaning` • `Visualization` • `Statistics`

---

## 04. Introduction to Machine Learning

Первый проект по машинному обучению.

Основные темы проекта:

- условная вероятность;
- теорема Байеса;
- ошибки первого и второго рода;
- обработка текстовых данных;
- построение признаков (Bag of Words и TF–IDF);
- обучение наивного байесовского классификатора;
- анализ качества модели;
- исследование устойчивости модели к атаке "отравления".

**Полученные навыки**

`Machine Learning` • `Naive Bayes` • `NLP` • `Scikit-Learn` • `Model Evaluation`
